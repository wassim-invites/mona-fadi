Hosting on GitHub Pages
✦ ✦ ✦

Follow these steps to host your invitation online for free — so the link works on any phone anywhere in the world.

✦ Step-by-Step Guide
1.

Create a GitHub Account

Go to github.com and sign up for a free account. Remember your username.

2.

Create a New Repository

Click + → New repository. Name it wedding. Set to Public. Check "Add a README file". Click Create repository.

3.

Upload Your Files

Click Add file → Upload files. Upload invite.html, admin.html, and index.html. Click Commit changes.

4.

Enable GitHub Pages

Go to Settings → Pages. Under Source, select Deploy from a branch → main / (root). Click Save.

5.

Your Site is Live!

After 1–2 minutes:
https://YOUR-USERNAME.github.io/wedding/admin.html
https://YOUR-USERNAME.github.io/wedding/invite.html

6.

Add Photos via GitHub

In your repo, click into the photos/ folder → Add file → Upload files. Upload your photos. Then in Settings, type the relative path like photos/church.png in the URL field — the preview will appear automatically.

7.

Generate Links

Open your admin panel URL. Go to Settings, fill in everything, click Save. Then go to Generate, enter a guest name, copy the link and send via WhatsApp. Each link contains all your wedding settings embedded — works on any phone.

✦ Music tip: Upload your .mp3 file to the photos/ folder on GitHub and use the path photos/music.mp3 in the Music field.

✦ Cross-Device RSVP Setup (5 minutes — highly recommended)

This makes every guest's RSVP appear in your admin dashboard instantly — even when they submit from their own phone.

1.

Create GitHub Personal Access Token

Go to github.com → Settings → Developer settings → Personal access tokens → Fine-grained tokens.
Click Generate new token. Set:
• Name: Wedding Settings
• Expiration: No expiration (or 1 year)
• Repository access: Only select repositories → choose your wedding repo
• Permissions → Contents: Read and write
Click Generate token. Copy it immediately — it won't show again.

2.

Paste in Admin Settings

Go to Settings tab → find GitHub Repository and GitHub Personal Access Token.
Enter your repo as toufic-reine/wedding and paste the token.
Click Save & Sync to Cloud ✦

3.

Paste This Code

Click the box below and press Ctrl+A then Ctrl+C to copy all:

4.

Deploy as Web App

Click Deploy → New deployment → Web app. Set Execute as: Me. Set Who has access: Anyone. Click Deploy. Copy the Web app URL.

5.

Paste URL in Admin — Done!

Go to Settings → Google Apps Script URL. Paste the Web app URL. Click Save All Settings.

From now on: every time you save settings or generate a guest link, it auto-syncs to Google Sheets. Guests open short clean links. RSVPs appear in your dashboard automatically. No file uploads ever needed.
