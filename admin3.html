Hosting Guide
— GitHub Pages setup —
✦ Setup Steps
1.

Create GitHub Account

Go to github.com and sign up.

2.

Create Repository

Click + → New repository. Name it (e.g. fadi-mona). Set to Public. Check "Add a README". Create.

3.

Upload Files

Upload invite3.html and admin3.html. Commit.

4.

Enable GitHub Pages

Settings → Pages → Deploy from branch → main / (root) → Save.

5.

Your Live URLs

https://USERNAME.github.io/fadi-mona/admin3.html
https://USERNAME.github.io/fadi-mona/invite3.html

6.

Create GitHub Token

Settings → Developer settings → Fine-grained tokens. Repository = fadi-mona. Permissions → Contents: Read and write. Paste in Settings above.

Photos: Create a photos/ folder in your repo and use paths like photos/hero.jpg in Settings.
